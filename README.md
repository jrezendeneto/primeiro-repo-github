# Meu Primeiro Repositório

Este é um repositório de exemplo para eu aprender o fluxo de Pull Request no GitHub.

## Sobre

Esse projeto foi criado com a ajuda do Claude Code para praticar:

- Criar um repositório
- Fazer um commit
- Abrir uma branch
- Enviar um Pull Request

## Como contribuir

Se vocë quiser contribuir, basta abrir um Pull Request com sua sugestão.
